## Descripción

Esta plataforma de e-commerce fue desarrollada como solución integral para pequeños y medianos comercios que buscan digitalizar sus ventas. La arquitectura se basa en microservicios para garantizar escalabilidad.

## Características principales

- 🛒 Carrito de compras con persistencia
- 💳 Integración con Stripe para pagos
- 📊 Dashboard de administración
- 🔍 Búsqueda con filtros avanzados
- 📱 Diseño totalmente responsive

## Stack técnico

El frontend fue construido con **React** y **TypeScript**, mientras que el backend utiliza **Node.js** con **Express**. La base de datos es **PostgreSQL** y el despliegue se realiza con **Docker** en AWS.
